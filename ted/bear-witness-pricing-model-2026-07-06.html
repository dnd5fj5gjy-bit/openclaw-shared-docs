Bear Witness · Internal Pricing Model

Priced on evidence, not estimate

The Strategy deck left Channel Management, Growth Retainer and Full Marketing Function priced as "figures aren't locked yet." This is the model that locks them - real Claude API cost, real infrastructure, real oversight time, checked against what a client would otherwise pay.

See the model
Why this exists

A starting price needs a floor and a ceiling. This builds both.

The floor is cost-to-serve: what it actually costs Bear Witness to run a client's Claude Sonnet 5 / Opus 4.8 agent stack for a month, plus the infrastructure and oversight time behind it. The ceiling is the client's counterfactual: what they'd otherwise pay in salary, on-costs, or agency fees for the same output. A defensible starting price sits well above the floor and meaningfully below the ceiling - not a guess pulled from a competitor's rate card.

This is a starting-point estimate, not a locked number. Every figure below is built from a stated assumption Olly or Jesse can challenge individually - real cost-to-serve gets refined the moment a real paying client is running, exactly as flagged in the Strategy deck's commercials section.

02 · Fixed costs - hosting, hardware, oversight

Infrastructure that gets cheaper per client as the client base grows.

Two of the three costs here are shared across every client, not billed per client - which is exactly the "more clients without more headcount" logic this whole model is built to demonstrate.

Hosting / infrastructure

Multi-tenant rented cloud VM

Superseded assumption: this deck previously modelled hosting as a shared container platform at £10-20/client/month alongside a separately-amortised Mac Studio. The Cloud Hosting & Monetisation deck validated a cleaner, cheaper real-world model instead - rented Linux VMs (Hetzner CX22 £3.26/mo, DigitalOcean/AWS Lightsail £18.96/mo), each hosting up to 8 isolated client agent workspaces, software-isolated per client the same way Ted already serves multiple Telegram users off one machine. No physical hardware is owned or amortised at all.

Hetzner CX22 (8 clients/box)£0.41/client/mo
DigitalOcean/AWS (8 clients/box)£2.37/client/mo
Used in totals below (conservative)£2.37/mo
Mac Studio - retired as the scaling model

Was capex; now Phase 1 of a two-phase plan

The Mac Studio is BGV's own machine and still runs agents internally - but it is no longer the thing client capacity is modelled against. An agent is a Claude Code CLI process calling Anthropic's API; it needs no local GPU and no specific physical box, so there is no ceiling to amortise against and no "buy a second machine at N clients" threshold to plan around. Phase 1 (now) is rented multi-tenant cloud VMs, opex-only; Phase 2 (post-fundraise) is BGV's own data centre, once traction justifies the capital - full detail in the Cloud Hosting & Monetisation deck.

04 · Sensitivity - quiet, average, heavy client

A single point estimate invites "what if usage is higher than that." This answers it.

Each tier modelled at three usage scenarios: a quiet client (60% of the baseline turn volume used throughout this deck), the average client (100% - the baseline figure used everywhere above), and a heavy client (150% of baseline, with oversight hours scaled up too, since a heavier client also needs more human review). Claude API compute scales directly with turn volume; multi-tenant cloud hosting is held flat at £2.37/client in every scenario, since a rented VM's box cost doesn't change with how many turns a client runs through it - only the number of clients per box matters, and that's already covered in section 02b; oversight hours scale but less than linearly.

TierQuiet client (60%)Average client (100%)Heavy client (150%)
Channel Management£118/mo£194/mo£320/mo
Growth Retainer£329/mo£547/mo£900/mo
Full Marketing Function£876/mo£1,458/mo£2,386/mo

"Average client" figures now match the section 03 headline totals exactly, since hosting is a flat £2.37/client line rather than a usage-metered one - one less place for two "correct but different" numbers to diverge. Quiet-client oversight scaled at the same 60% as turn volume; heavy-client oversight assumed at roughly 1.7x average hours (5 / 14 / 34 hrs/month vs. 3 / 8 / 20 baseline) to reflect the extra review a high-volume account actually needs, not just more API calls.

Even the heavy-client scenario stays well under the recommended starting price at every tier - £320 vs. a £950-1,200 Channel Management price, £900 vs. £2,500-3,000 for Growth Retainer, £2,386 vs. £6,000-8,000 for Full Marketing Function. That's the actual answer to "what if usage runs hotter than modelled": the margin compresses, but it doesn't disappear - and with hosting now a flat, negligible line rather than a usage-metered one, this scenario is if anything more conservative than the previous version of the model.

05 · The client's counterfactual

What the client would otherwise pay to solve the same problem.

UK benchmarks: average marketing manager salary runs £39,000-£45,000/year (Indeed £42,358, Glassdoor £44,511, PayScale £39,438), plus roughly 20-30% in employer on-costs (NI, pension, benefits) - call it £52,000-£56,000/year fully loaded, or about £4,400-£4,700/month. UK agency retainers: single-channel work £1,250-£3,500/month; full-service retainers £3,500-£16,750/month; SMEs typically start at £1,000-£2,000/month. A full marketing/competitor audit from a traditional agency still runs £10,000-£15,000 over 4-6 weeks; a standard SME audit £1,000-£5,000 (both figures reused unchanged from the Strategy deck).

06 · Competitor benchmark

Bear Witness against both the old world and the emerging one.

The counterfactual above compares Bear Witness to traditional in-house and agency costs. The more relevant comparison going forward is the emerging "AI marketing agency" / AI-run retainer category, since that's who a prospective client will actually shop against once they know AI-delivered marketing is an option. 2026 market data on that category: AI SEO/content retainers run $2,000-$20,000/month (market average ~$3,200); AI-assisted paid media management runs $3,000-$8,000/month for a focused scope; comprehensive AI marketing retainers spanning SEO, paid, content and automation cluster $2,000-$8,000/month for small-to-mid clients, rising toward $15,000-$20,000/month for the top end. Separately, UK fractional CMO engagements - a different but adjacent category - run £3,000-£10,000/month retainers (4-8 days/month) or £750-£2,500/day, rising to £2,500+/day for specialist FinTech/SaaS work.

07 · The margin, and the recommendation

A starting price for each tier, with the multiple shown.

Each recommendation sits at roughly 4-6x cost-to-serve, and comfortably below the client's counterfactual - a real margin, not a guess, and one that gets sharper the moment a paying client sets the actual cost-to-serve.

The recommended price bands are unchanged: £950-1,200 / £2,500-3,000 / £6,000-8,000 per month still hold. Hosting was always a small share of cost-to-serve, so the lower, multi-tenant-cloud figure moves margin, not the recommendation. What does change is the caveat behind it: this deck used to carry an implicit "assuming BGV can afford enough Mac Studios to serve the book" condition on those prices, because cost-to-serve was tied to owned hardware with a real ceiling. That condition is gone. The price bands now rest on a cost floor that scales in £3-19/month steps indefinitely, with no capital purchase required at any client count modelled - the margin story is stronger, and there is no scaling caveat left to flag.
