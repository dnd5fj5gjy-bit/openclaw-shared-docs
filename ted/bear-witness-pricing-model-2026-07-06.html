Bear Witness · Internal Pricing Model

Numbers, not vibes

The Strategy deck left Channel Management, Growth Retainer and Full Marketing Function priced as "figures aren't locked yet." This is the model that locks them - real Claude API cost, real infrastructure, real oversight time, checked against what a client would otherwise pay.

See the model
Why this exists

A starting price needs a floor and a ceiling. This builds both.

The floor is cost-to-serve: what it actually costs Bear Witness to run a client's Claude Sonnet 5 / Opus 4.8 agent stack for a month, plus the infrastructure and oversight time behind it. The ceiling is the client's counterfactual: what they'd otherwise pay in salary, on-costs, or agency fees for the same output. A defensible starting price sits well above the floor and meaningfully below the ceiling - not a guess pulled from a competitor's rate card.

This is a starting-point estimate, not a locked number. Every figure below is built from a stated assumption Olly or Jesse can challenge individually - real cost-to-serve gets refined the moment a real paying client is running, exactly as flagged in the Strategy deck's commercials section.

01 · Cost to serve - Claude API compute

The single biggest real cost, modelled per tier.

02 · Fixed costs - hosting, hardware, oversight

Infrastructure that gets cheaper per client as the client base grows.

Two of the three costs here are shared across every client, not billed per client - which is exactly the "more clients without more headcount" logic this whole model is built to demonstrate.

Hosting / infrastructure

Shared container platform

The Agent Workforce deck's own estimate: infra cost stays "tens to low hundreds £/mo at 10 clients" on a shared platform (e.g. Fly.io/Modal), well below Claude API usage cost. Scaled per client, that's roughly £10-20/month depending on tier weight - kept here as the working assumption for the totals below, now checked against real published pricing in the next panel.

Mac Studio / core compute

Existing infrastructure, amortised

Apple lists the Mac Studio (M4 Max, 14-core CPU/32-core GPU, 36GB/512GB) from £2,099 on apple.com/uk; a higher-spec configuration suited to running several concurrent agent sessions runs to roughly £3,500-£4,000. This is already-owned hardware, not a new per-client cost - modelled here at an effective £3,700, amortised over 24 months (£154/month total), shared across the whole client book.

02a · Real infra pricing, not a range

The £10-20/client/month hosting line, checked against two published rate cards.

Two published models, both checked directly: Fly.io's per-second Machine pricing (an always-on container) and Modal's per-core-second / per-GiB-second pricing (a metered, pay-for-actual-use container). Both come in below the £10-20 working assumption used in the totals - which means that assumption has built-in headroom for bandwidth, storage, monitoring and the inevitable "we under-modelled something" margin, rather than being an optimistic guess.

Model A · Fly.io, always-on Machine

Persistent container, priced per second

Fly.io's published Amsterdam-region rates: shared-cpu-1x/256MB = $0.0028/hr ($2.02/mo run continuously); shared-cpu-1x/1GB = $0.0082/hr ($5.92/mo); shared-cpu-1x/2GB = $0.0154/hr ($11.11/mo). At $1=£0.79, a 1GB container run 24/7 all month costs £4.68; a 2GB container costs £8.78.

By roughly 20 clients the Mac Studio line has fallen under 1.5% of cost-to-serve and keeps shrinking - past that point, cost-to-serve is effectively pure variable cost (Claude API + hosting + oversight), and adding client #51 costs almost exactly what client #21 did. That's the actual mechanism behind "this scales without re-buying hardware or hiring," not just an assertion.

04 · Sensitivity - quiet, average, heavy client

A single point estimate invites "what if usage is higher than that." This answers it.

Each tier modelled at three usage scenarios: a quiet client (60% of the baseline turn volume used throughout this deck), the average client (100% - the baseline figure used everywhere above), and a heavy client (150% of baseline, with oversight hours scaled up too, since a heavier client also needs more human review). Claude API compute and metered infra (Model B, above) scale directly with turn volume; Mac Studio stays fixed at the 10-client rate; oversight hours scale but less than linearly.

TierQuiet client (60%)Average client (100%)Heavy client (150%)
Channel Management£142/mo£210/mo£332/mo
Growth Retainer£356/mo£567/mo£843/mo
Full Marketing Function£949/mo£1,488/mo£2,224/mo

"Average client" figures use metered infra (Model B) rather than the flat £10/15/20 hosting line, which is why they land a few pounds from the £215/£575/£1,491 headline totals above - both are correct, they're answering slightly different questions (flat assumption vs. usage-metered). Heavy-client oversight assumed at roughly 1.7x average hours (5 / 12 / 30 hrs/month vs. 3 / 8 / 20 baseline) to reflect the extra review a high-volume account actually needs, not just more API calls.

Even the heavy-client scenario stays well under the recommended starting price at every tier - £332 vs. a £950-1,200 Channel Management price, £843 vs. £2,500-3,000 for Growth Retainer, £2,224 vs. £6,000-8,000 for Full Marketing Function. That's the actual answer to "what if usage runs hotter than modelled": the margin compresses, but it doesn't disappear.

04 · The client's counterfactual

What the client would otherwise pay to solve the same problem.

UK benchmarks: average marketing manager salary runs £39,000-£45,000/year (Indeed £42,358, Glassdoor £44,511, PayScale £39,438), plus roughly 20-30% in employer on-costs (NI, pension, benefits) - call it £52,000-£56,000/year fully loaded, or about £4,400-£4,700/month. UK agency retainers: single-channel work £1,250-£3,500/month; full-service retainers £3,500-£16,750/month; SMEs typically start at £1,000-£2,000/month. A full marketing/competitor audit from a traditional agency still runs £10,000-£15,000 over 4-6 weeks; a standard SME audit £1,000-£5,000 (both figures reused unchanged from the Strategy deck).

06 · Every assumption, in one place

Assumptions box - flag any of these to update the model
  • FX rate: $1 = £0.79, applied to all USD API costs.
  • Sonnet 5 pricing: $3/$15 per million input/output tokens (standard rate used, not the $2/$10 intro rate that expires 31 Aug 2026).
  • Opus 4.8 pricing: $5/$25 per million input/output tokens.
  • Cache read discount: ~0.1x the input rate.
  • Cache write premium: ~1.25x the input rate, 5-minute TTL.
  • Cache mix per turn: 60% cached input, 30% fresh input, 10% cache write - modelled the same across all three tiers.
  • Channel Management volume: 200 turns/day, 2,000 input + 400 output tokens/turn, 100% Sonnet 5.
  • Growth Retainer volume: 500 turns/day, 2,500 input + 500 output tokens/turn, 85% Sonnet 5 / 15% Opus 4.8.
  • Full Marketing Function volume: 1,200 turns/day, 3,000 input + 600 output tokens/turn, 70% Sonnet 5 / 30% Opus 4.8.
  • Days per month: 30, agents run daily including weekends.
  • Hosting cost: £10-20/client/month, reused unchanged from the Agent Workforce deck's "tens to low hundreds £/mo at 10 clients" estimate.
  • Mac Studio cost: £3,700 effective spec (Apple UK lists the base M4 Max from £2,099; a higher-memory config for concurrent sessions runs £3,500-4,000), amortised over 24 months, shared across the whole client book.
  • Oversight hours: 3 / 8 / 20 hours per month for Channel Management / Growth Retainer / Full Marketing Function.
  • Oversight rate: £50/hour notional internal rate - BGV's own cost of time, not a market billing rate.
  • Client counterfactual: UK marketing manager salary £39-45k/year plus 20-30% on-costs; UK agency retainers £1,250-16,750/month depending on scope; audit figures reused unchanged from the Strategy deck.
