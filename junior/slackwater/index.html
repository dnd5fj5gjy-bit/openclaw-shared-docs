How the constraint logic works

Layer 1 — Environmental

Wind speed, tidal windows, and temperature thresholds auto-fetched daily from Met Office DataPoint and the ADMIRALTY tidal API using your site's grid reference. Each asset can have different limits — the system tracks them all, and you never type a tide time again.

Layer 2 — Regulatory

Permit to work expiry. EA consents and gate schedules. Third-party access windows. Method statement scope. All extracted from documents automatically — no manual data entry.

Layer 3 — Personnel

CTC clearance levels per worker. Certification expiry tracking. Minimum crew requirements per task type. ERT and safety-critical role checks before any mobilization is approved.

Layer 4 — Sequencing

Scaffold before engineers access. ERT on station before in-water work. Multi-contractor dependencies tracked. No one stands around waiting for the team before them.
