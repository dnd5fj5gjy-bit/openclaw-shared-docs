**Opening offer:** £1,500 (not £500 — the custom page suggests sophistication)  
**Maximum:** £5,000 (this will require Jesse's sign-off above £2K — call Jesse if the counter comes back above £2K)  
**Timeline:** GoDaddy brokers typically respond within 5-7 business days. Start today to have room before June 24.

### Option B — Sedo Buyer Enquiry (run in parallel)

List as a buyer enquiry on Sedo.com simultaneously. Sedo reaches a different network of brokers. Run both in parallel.

### Option C — Contingency: Direct WHOIS Outreach

If both broker approaches fail, file a WHOIS request with ICANN to reveal the registrant contact behind GoDaddy's privacy shield. This takes longer and reveals you're looking, but it's available.
