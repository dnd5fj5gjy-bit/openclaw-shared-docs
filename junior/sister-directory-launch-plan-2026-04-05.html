Public-web read

I couldn't extract much from the Instagram profile pages because Instagram serves a very thin shell to the fetch tools here. What I can say is that Sister Directory is public enough to have an Instagram handle and a site deployment trail, but the launch is probably not yet defined by one clear message or one strong acquisition loop.

Core strategic read

Sister Directory should be launched like a curated clean-beauty discovery utility, not like a generic list.

Positioning hypothesis

Sister Directory is the easiest way to discover, compare, and share the clean beauty and non-toxic products women actually want to trust.

Launch problem

  1. The site exists, but nobody knows why it matters.
  2. The site has content, but no clear repeatable acquisition engine.
  3. The Instagram exists, but it has not been converted into a traffic or referral loop.

Two-week launch plan

Week 1 — Fix the product story and launch system

  1. Lock the one-line promise.
  2. Make the homepage painfully clear.
  3. Create 3 content pillars.
  4. Turn Instagram into a traffic funnel.
  5. Build a simple referral loop.

Week 2 — Drive attention and prove value

  1. Launch with 10-15 high-quality featured entries.
  2. Run a founder/curator-led launch burst.
  3. Seed the network manually.
  4. Capture email or waitlist signup.
  5. Add a weekly reason to return.

Promotional strategy

Organic channels first

  • Instagram Reels and carousels
  • Founder-led posting
  • Stories with poll / question sticker behavior
  • Partner reposts
  • Community shares and referrals

Press / earned attention

Pitch it as a smart clean-beauty discovery tool or community resource. Use a strong angle and lean into a niche hard if the niche is clear.

Partnerships

Cross-promote with relevant women-led brands, clean beauty founders, wellness operators, and creators. Create featured slots for aligned brands. Make it feel like taste and access, not ads.

Paid only after the story works

Don't pour into Meta before the site converts organically. If paid is used, boost the best performing social proof posts and best list pages.

Role split

  • Ruby: run content, curation, and day-to-day social rhythm
  • Jesse: owner-level decisions, opening doors, strategic pushes, amplification
  • Junior: help keep the launch machine moving, write copy, track responses, surface what is working

My opinion

The mistake to avoid is making Sister Directory feel like a loose list of things. It should feel like a taste-led clean-beauty discovery layer with a strong editorial point of view.
