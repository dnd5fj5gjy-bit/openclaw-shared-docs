# Felix Nightly Reset Playbook

A concise operating template for the last 15 minutes of the day or the first 15 minutes of the morning. Designed to keep momentum without creating noise.

## Use when
- The day felt fragmented
- You have too many open loops
- You want a clean restart instead of more planning
- You need to protect tomorrow’s focus

## The 5-minute triage
1. Write the 3 highest-value outcomes for tomorrow.
2. Mark anything stale, delegated, or irrelevant as parked.
3. Identify one thing that would make tomorrow easier if done now.
4. Decide the first task for tomorrow before you stop.
5. Close the notebook/app and stop negotiating with yourself.

## The rule set
- If it takes under 2 minutes, do it now.
- If it takes under 20 minutes and removes friction, do it now.
- If it creates more than 3 new tasks, postpone unless urgent.
- Do not “organize” instead of deciding.
- Do not start a new strategy rabbit hole after 9 PM.

## Tomorrow-start script
Copy this into a note:

> Tomorrow’s win is: __________
>
> First action: __________
>
> One thing to ignore: __________
>
> If I get distracted, I will return to: __________

## Team-use version
For Junior / Nexus / BGV coordination:
- Outcome: one sentence
- Owner: one name
- Next check-in: one time
- Blocker: one line

Anything longer becomes a meeting disguised as a note.

## Why this works
It reduces decision fatigue, preserves momentum, and makes the next session easier to start than to avoid.
