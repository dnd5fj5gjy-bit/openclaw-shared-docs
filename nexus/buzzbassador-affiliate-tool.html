Buzzbassador Bulk Affiliate Link Generator

Built by Nexus · April 2026 · Tool: workspace/tools/buzzbassador_gen.js

What this does: Turns any list of product URLs into affiliate links — in seconds, not minutes. One browser login to extract your account token, then all links are built locally without clicking through the UI per product.

The Problem It Solves

The Buzzbassador Direct Product Link Generator is a one-at-a-time UI tool. For 79 Fat and the Moon products, that meant:

Method Time (79 products) Error risk
Manual UI (paste → generate → copy, repeat) ~8–12 minutes High — easy to miss one or copy wrong
This tool (bulk) ~20 seconds Zero — deterministic from URL list

More importantly: the next time Sister Directory onboards a new brand with 50+ products, this is already done.

How It Works

The affiliate link format is deterministic once you know three account-specific values:

  • Shopify domain — e.g. fat-the-moon.myshopify.com
  • Discount code — e.g. SISTER
  • buzz_ref token — e.g. 50361849673VztU (tied to your member account)

The tool extracts these by logging in once and generating a single test link. After that, all 79+ links are constructed locally:

https://{shopify_domain}/discount/{CODE}?buzz_ref={TOKEN}&redirect={product_path}

Variant parameters (e.g. ?variant=39591943143459) are automatically stripped — same behaviour as the UI generator.

Quick Start

Step 1 — Prepare your URL list

Create a plain text file with one URL per line. Names are optional but nice:

Acid Base https://www.fatandthemoon.com/collections/first-aid-1/products/acid-base
All Cream https://www.fatandthemoon.com/collections/masks-of-the-god-dess/products/all-cream
Deodorant Cream https://www.fatandthemoon.com/collections/deo/products/deodorant-cream?variant=19566024515

Step 2 — Run (first time — logs in, caches token)

node ~/agents/nexus/workspace/tools/buzzbassador_gen.js --email EMAIL --password PW --urls-file urls.txt --format csv > output.csv

Step 3 — Run again (no login needed — uses cached token)

node ~/agents/nexus/workspace/tools/buzzbassador_gen.js --email EMAIL --password PW --use-cached-token --urls-file urls.txt --format csv > output.csv

Using With a New Brand

Each Buzzbassador member account has its own buzz_ref token and discount code. To onboard a new brand:

  1. Get the brand's Buzzbassador member login credentials
  2. Use --no-cache (or delete the cache file) so it logs in fresh
  3. Provide a --test-url that's a valid product URL for that brand's store
  4. The tool extracts the new account's token and generates all links
node ~/agents/nexus/workspace/tools/buzzbassador_gen.js --email newbrand@email.com --password PW --no-cache --test-url https://newbrand.com/products/any-product --urls-file newbrand_urls.txt --format csv
