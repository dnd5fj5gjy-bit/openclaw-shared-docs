# Nightly Creative Direction for Felix

## What this should do
Each night, produce something that is:
- genuinely useful,
- a little surprising,
- visibly moving Felix or the team toward a goal,
- and small enough to finish without friction.

## Best formats
1. **Short document**
   - strategy note
   - idea brief
   - decision memo
   - teardown of a tool, market, or workflow

2. **Small build**
   - simple website
   - interactive landing page
   - one-purpose internal tool
   - dashboard or visualizer

3. **Actionable output**
   - a cleaned-up checklist
   - a plan for a project
   - a concise playbook
   - a set of prompts or templates

## What to optimise for
- usefulness over novelty
- clarity over polish
- momentum over perfection
- things Felix can actually use the next day
- things the team can reuse

## Good nightly prompts
- Build something that helps Felix think faster.
- Make something the team could use tomorrow.
- Find one bottleneck and turn it into a tool.
- Write a compact brief on a valuable opportunity.
- Turn a messy idea into a clean artifact.

## Suggested default output rotation
- Mon: build
- Tue: doc
- Wed: tool or template
- Thu: research brief
- Fri: something fun but useful
- Sat: idea exploration
- Sun: weekly synthesis

## Rule of thumb
If the night’s idea can be explained in one sentence, it is probably the right size.

If not, make it smaller.
