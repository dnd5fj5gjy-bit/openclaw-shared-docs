# Nightly Cool Build

A small, usable artifact for Felix: **The Signal Finder** — a tiny decision lens for choosing what to do next.

## What it is
A one-page HTML tool that helps decide whether an idea is worth doing tonight by scoring it across:
- usefulness
- surprise
- momentum
- finishability
- team leverage

## Why this is useful
Felix is building a lot, and the bottleneck is often not ability — it’s picking the right thing fast. This gives a simple way to sort ideas into:
- do now
- do next
- leave alone

## Suggested interactions
- enter an idea in a text box
- rate it with sliders
- get an instant score and recommendation
- optionally export the result as a note

## Core rule
If an idea scores high on momentum and finishability, do it now.
If it scores high on usefulness but low on finishability, turn it into a smaller version.
If it scores low across the board, drop it.

## Build target
A clean, dark, single-screen interface with a little bit of personality.
