Quick Start — The One Command

/Users/bgvai/agents/nexus/workspace/tools/affiliate_pipeline.sh \ --store fatandthemoon.com \ --email Ruby@sisterdirectory.com \ --password Moonfat10! \ --use-cached-token \ --format csv > fat-moon-all-links.csv

That's it. Opens no browser. Pulls all 110 products. Generates all affiliate links. Saves to CSV.

Common Recipes

All products, CSV output

./affiliate_pipeline.sh --store fatandthemoon.com --email Ruby@sisterdirectory.com --password Moonfat10! --use-cached-token --format csv > all-links.csv

One collection only

./affiliate_pipeline.sh --store fatandthemoon.com --email Ruby@sisterdirectory.com --password Moonfat10! --use-cached-token --collection adornments --format csv > adornments-links.csv

New brand — first time (no cached token)

./affiliate_pipeline.sh --store newbrand.com --email brand@email.com --password PW --format csv > newbrand-links.csv

First run logs into Buzzbassador and caches the token. Subsequent runs use --use-cached-token.

Just get product URLs (no Buzzbassador)

python3 shopify_products.py --store fatandthemoon.com --format csv
